
1. touch bash1.txt - Предварительно создайте файл bash1.txt, в который вы будете добавлять выполненные команды.

2. cd - Открыть домашнюю директорию через терминал

3. pwd - Определить имя папки, в которой вы находитесь

4. mkdir test1 - Создать внутри этой папки каталог с именем test1

5. cd test1 - Перейти в папку test1

6. touch 1 2 3 - Создать файл 1,2 и 3 внутри каталога test1

7. ls - Проверить содержимое каталога test1

8. cd - Перейти в домашнюю директорию

9. mkdir test2 - Создать папку test2 внутри домашней директории

10. rmdir test2 - Удалить папку test2


11. rm ~/test1/2 - Удалить файл 2 из папки test1

12. mkdir test3 - Создать папку в домашней директории test3 и добавить в нее два файла

13. rm -r test3 - Удалить папку test3

14. mkdir test4 - Создать папку test4 в домашней директории

15. mv ~/test1/1 ~/test1/3 ~/test4/ - Переместить файлы 1 и 3 из папки test1 в папку test4

16. echo -e "line\nline\nline" > ~/test4/1  - Добавить в файл 1 три строки со словами line

17. cat ~/test4/1  - Посмотреть содержимое файла 1

18. echo -e "line\nline\nline" > ~/test4/3 - Добавьте в файл 3 три строки со словами line

19. cat ~/test4/1 ~/test4/3 - Просмотрите содержимое двух файлов (1 и 3) сразу

20. nano ~/test4/1 - Используя один из редакторов замените все строки в файле 1
